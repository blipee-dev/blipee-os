Welcome to the Dashboard Template

This template demonstrates how to use the Blipee component system. The navbar and sidebar are automatically loaded from separate component files, making it easy to maintain consistency across all your dashboards.


Key Features:

  • Reusable navbar and sidebar components
  • Automatic theme toggle (light/dark mode)
  • Responsive sidebar with collapse/expand
  • Shared CSS variables for consistent styling
  • localStorage persistence for user preferences
  • Animated gradient background

Getting Started

To create a new dashboard based on this template:

  1. Copy this file and rename it (e.g., my-dashboard.html)
  2. Update the title in the <head>
  3. Add your dashboard-specific styles in the <style> block
  4. Replace the content inside <main id="mainContent">
  5. Add a navigation link in components/navbar.html if needed

Check the COMPONENTS_GUIDE.md file for detailed documentation.
